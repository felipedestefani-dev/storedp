# Random Studio (Next.js)

Um site aleatório feito com Next.js (App Router) que gera cor, frase, número e “vibe”, com histórico salvo no navegador.

## Rodar local

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deploy na Vercel

- Importe o repositório na Vercel
- Build Command: `npm run build`
- Output: `.next` (detectado automaticamente)
